## FAQ

### Why do I see no manga?
Booklore is a self-hosted comic/manga media server.

### Where can I get more information about Booklore?
You can visit the [Booklore](https://github.com/booklore-app/booklore) website for for more information.

### The Booklore extension stopped working?
Make sure that your Booklore server and extension are on the newest version.

### Can I add more than one Booklore server or user?
Yes, currently you can add up to 11 different Booklore instances to the app. The number of instances
available can be customized in extension settings.

### Can I test the Booklore extension before setting up my own server?
Yes, you can try it out with the DEMO server `https://demo.booklore.org`, username `demo@booklore.org` and
password `booklore-demo`.

### Why are EPUB chapters/books missing?
Mihon/Tachiyomi does not support reading text. EPUB files containing only images will be available
if your Booklore server version is [1.9.0](https://github.com/gotson/booklore/releases/tag/1.9.0)
or newer.

## Guides

### How do I add my Booklore server to the app?
Go into the settings of the Booklore extension (under Browse -> Extensions) and fill in your server
address and login details.
